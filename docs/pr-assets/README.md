# Stellarator PR assets

This directory contains reviewer-facing figures for the stacked stellarator geometry PR series.

Each subdirectory matches one PR:
- `pr1/`: package-backed stellarator geometry foundation
- `pr2/`: package-backed vs legacy kernel parity
- `pr3/`: DESC-native Boozer and flux-tube support
- `pr4/`: provider API and resolved DESC grid reporting
- `pr5/`: maintained workflow example
- `pr6/`: maintained vs legacy example split
- `pr7/`: provider/package parity gates
- `pr8/`: geometry comparison utility

These assets are meant for PR descriptions and review discussions. They are not runtime inputs to DREAM.
