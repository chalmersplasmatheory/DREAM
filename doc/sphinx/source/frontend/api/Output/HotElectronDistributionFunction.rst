HotElectronDistributionFunction
===============================
The ``HotElectronDistributionFunction`` encapsulates the output of the hot
electron distribution function :math:`f_{\rm hot}` obtained from DREAM. The
``HotElectronDistributionFunction`` class is derived from the
:ref:`do-distributionfunction` class, which is in turn derived from the
:ref:`do-kineticquantity` class.
